# howToSubmit 模板示例

常见外链平台的 `howToSubmit` 字段模板。

---

## 目录站 (Directory)

### 简单表单提交

```markdown
## 提交方式

1. 访问 https://example-directory.com/submit
2. 无需登录
3. 填写表单：
   - **Website Name**: 填写网站名称
   - **URL**: 填写完整网址（含 https://）
   - **Description**: 填写 100-200 字英文描述
   - **Category**: 选择 "Tools" 或 "SaaS"
   - **Email**: 填写联系邮箱
4. 点击 "Submit" 按钮
5. 检查邮箱确认链接

## 注意事项
- 描述不接受中文
- 审核周期约 3-7 天
```

### 需要注册

```markdown
## 提交方式

1. 访问 https://startup-directory.com/register
2. 注册账号（需要邮箱验证）
3. 登录后访问 /submit
4. 填写表单：
   - **Startup Name**: 网站名称
   - **Website**: 网址
   - **Tagline**: 一句话介绍
   - **Description**: 详细介绍
5. 点击 "Submit for Review"

## 注意事项
- 必须登录
- 跳过此资源或手动处理
```

## 论坛 (Forum)

```markdown
## 提交方式

1. 访问 https://forum.example.com/c/show-and-tell
2. 需要登录账号
3. 点击 "New Topic"
4. 填写：
   - **Title**: "[Show] 网站名称 - 简短描述"
   - **Content**: 问题描述 + 解决方案 + 链接
5. 点击 "Post"

## 内容模板
标题: [Show] MyBacklinks - Backlink Management

大家好！我做了一个外链管理工具...

链接: https://mybacklinks.app

## 注意事项
- 需要账号
- 避免过度营销
```

## 博客投稿 (Guest Post)

```markdown
## 提交方式

1. 访问 https://blog.com/write
2. 需要登录
3. 撰写 800+ 字技术文章
4. 在文章中自然插入 1-2 个链接
5. 点击 "Submit for Review"

## 内容要求
- 原创技术内容
- 链接需自然融入

## 注意事项
- 审核周期 1-3 天
```

## 通用模板

```markdown
## 提交方式

1. 访问 [URL]
2. [登录要求]
3. 填写表单：
   - **[字段]**: [说明]
4. [提交操作]

## 注意事项
- [限制]
```
