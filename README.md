## Quick Start

### Install the MCP Server

```bash
npx mybacklinks-mcp setup
```

This will output configuration you can add to your AI tool (Claude Code, Cursor, or Kiro).

### Use with AI Assistants

Once configured, you can ask your AI assistant to:

- "List all my projects"
- "Show pending backlinks for my SaaS project"
- "Add a new backlink from producthunt.com"
- "Analyze my SEO performance this month"
